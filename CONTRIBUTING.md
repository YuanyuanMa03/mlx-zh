# 贡献指南

感谢你对 MLX-LM 中文指南的关注！

## 如何贡献

### 报告问题

如果你发现文档中的错误或有改进建议：

1. 在 [Issues](https://github.com/YOUR_USERNAME/mlx-lm-zh-guide/issues) 中搜索是否已有类似问题
2. 如果没有，创建新的 Issue，详细描述问题或建议

### 提交文档改进

1. **Fork 本仓库**
2. **创建分支** (`git checkout -b feature/your-feature`)
3. **提交修改** (`git commit -m 'Add some feature'`)
4. **推送分支** (`git push origin feature/your-feature`)
5. **创建 Pull Request**

### 文档规范

- 使用简体中文编写
- 代码示例需要可运行
- 添加适当的标题层级
- 保持与原 MLX-LM 项目的一致性

### 文档结构

```
mlx-lm-zh-guide/
├── README.md           # 主页
├── CONTRIBUTING.md     # 贡献指南（本文件）
├── LICENSE            # MIT 许可证
├── docs/              # 详细文档
├── examples/          # 代码示例
└── .github/           # GitHub 配置
```

## 代码示例要求

- 添加必要的注释
- 使用推荐的中文模型作为示例
- 包含预期的输出说明

## 行为准则

- 尊重所有贡献者
- 建设性反馈
- 专注于改进文档质量

## 许可证

贡献的内容将采用与本项目相同的 MIT 许可证。
