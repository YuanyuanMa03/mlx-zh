# 系统要求

## 硬件要求

### Mac 机型

MLX-LM 只支持搭载 **Apple 芯片** 的 Mac 电脑：

- **M1** 系列：M1、M1 Pro、M1 Max、M1 Ultra
- **M2** 系列：M2、M2 Pro、M2 Max、M2 Ultra
- **M3** 系列：M3、M3 Pro、M3 Max、M3 Ultra
- **M4** 系列：M4、M4 Pro、M4 Max

> **注意**：Intel 芯片的 Mac 不支持 MLX-LM。

### 内存建议

不同规模的模型对内存的需求：

| 模型规模 | 精度 | 最低内存 | 推荐内存 |
|----------|------|----------|----------|
| 3B-4B | FP16 | 8GB | 16GB |
| 7B-8B | 4-bit 量化 | 16GB | 32GB |
| 7B-8B | FP16 | 32GB | 64GB |
| 13B-14B | 4-bit 量化 | 32GB | 64GB |
| 30B+ | 4-bit 量化 | 64GB | 128GB |

**推荐配置**：
- **入门级**：16GB 内存 + 7B 4-bit 模型
- **进阶级**：32GB 内存 + 14B 4-bit 模型
- **专业级**：64GB+ 内存 + 30B+ 模型

## 软件要求

### 操作系统

- **最低要求**：macOS 12.0 (Monterey)
- **推荐版本**：macOS 15.0 (Sequoia) 或更高
- **大模型优化**：需要 macOS 15.0+

### Python

- **版本**：Python 3.8 或更高
- **推荐**：Python 3.10 或 3.11

### 包管理器

可选择以下任一方式：

- **pip**（推荐）
- **conda**
- **Homebrew**（用于安装 Python）

## 如何查看你的 Mac 配置

### 查看芯片型号

```bash
sysctl -n machdep.cpu.brand_string
```

### 查看内存大小

```bash
# 方式1：系统信息
system_profiler SPHardwareDataType | grep Memory

# 方式2：查看物理内存
sysctl -n hw.memsize
```

### 查看 macOS 版本

```bash
sw_vers
```

## 性能优化建议

### 1. 升级到最新 macOS

如果你的 Mac 支持，升级到 macOS 15.0+ 可以获得大模型优化功能。

### 2. 增加有线内存限制（macOS 15+）

运行大模型时，可能需要增加系统有线内存限制：

```bash
# 查看当前限制
sysctl iogpu.wired_limit_mb

# 临时增加（需要 sudo）
sudo sysctl iogpu.wired_limit_mb=24000
```

> 警告：设置的值应大于模型大小但小于机器总内存。

### 3. 关闭不必要的应用

运行大模型前关闭内存密集型应用（如 Xcode、大型浏览器等）。

## 兼容性问题

### 不支持的设备

- ❌ Intel 芯片 Mac
- ❌ iPhone/iPad
- ❌ Windows/Linux 电脑（虽然有 MLX 的非 macOS 版本，但本指南专注于 Mac）

### 可能的性能问题

- **M1/M2 基础型号**：内存带宽较低，大模型推理较慢
- **8GB 统一内存**：只能运行小型模型（3B-4B）或使用极端量化
