# 命令行工具详解

MLX-LM 提供了丰富的命令行工具，让你无需编写代码即可使用大语言模型。

## 查看帮助

所有命令都支持 `--help` 参数查看详细帮助：

```bash
mlx_lm.generate --help
mlx_lm.chat --help
mlx_lm.convert --help
```

## generate - 文本生成

### 基本用法

```bash
mlx_lm.generate --prompt "你的提示词"
```

### 常用参数

| 参数 | 说明 | 示例 |
|------|------|------|
| `--model` | 指定模型 | `--model Qwen/Qwen2.5-7B-Instruct` |
| `--prompt` | 输入提示词 | `--prompt "你好"` |
| `--max-tokens` | 最大生成 token 数 | `--max-tokens 512` |
| `--temp` | 温度参数 (0-1) | `--temp 0.7` |
| `--top-p` | Top-p 采样 | `--top-p 0.9` |
| `--top-k` | Top-k 采样 | `--top-k 50` |
| `--seed` | 随机种子 | `--seed 42` |
| `--verbose` | 显示详细信息 | `--verbose` |

### 实用示例

```bash
# 中文对话
mlx_lm.generate \
    --model mlx-community/Qwen2.5-7B-Instruct-4bit \
    --prompt "请用三句话介绍机器学习"

# 创意写作
mlx_lm.generate \
    --prompt "写一首关于人工智能的七言诗" \
    --max-tokens 200 \
    --temp 0.9

# 代码生成
mlx_lm.generate \
    --prompt "用Python写一个快速排序算法" \
    --temp 0.2

# 使用随机种子确保可重复性
mlx_lm.generate \
    --prompt "解释什么是深度学习" \
    --seed 123 \
    --temp 0.7
```

## chat - 交互式聊天

### 启动聊天

```bash
mlx_lm.chat
```

### 指定模型聊天

```bash
mlx_lm.chat --model Qwen/Qwen2.5-7B-Instruct
```

### 聊天参数

| 参数 | 说明 |
|------|------|
| `--model` | 指定模型 |
| `--max-tokens` | 每次回复的最大 token 数 |
| `--temp` | 温度参数 |
| `--history-file` | 保存聊天历史 |

### 聊天技巧

```bash
# 使用中文模型获得更好的中文理解
mlx_lm.chat --model mlx-community/Qwen2.5-7B-Instruct-4bit

# 设置较低的 temperature 获得更稳定的回复
mlx_lm.chat --model Qwen/Qwen2.5-7B-Instruct --temp 0.5
```

## convert - 模型转换

### 转换 Hugging Face 模型为 MLX 格式

```bash
# 基本转换
mlx_lm.convert --model Qwen/Qwen2.5-7B-Instruct

# 转换并量化
mlx_lm.convert --model Qwen/Qwen2.5-7B-Instruct -q

# 指定输出路径
mlx_lm.convert \
    --model Qwen/Qwen2.5-7B-Instruct \
    --mlx-path ./my_qwen_model

# 上传到 Hugging Face
mlx_lm.convert \
    --model Qwen/Qwen2.5-7B-Instruct \
    -q \
    --upload-repo your-username/qwen2-7b-4bit
```

### 量化选项

```bash
# 4-bit 量化（默认）
mlx_lm.convert --model MODEL -q

# 指定量化位数
mlx_lm.convert --model MODEL -q --q-bits 4
```

## lora - 模型微调

详见 [模型微调教程](./05-模型微调.md)

```bash
# 基本微调
mlx_lm.lora \
    --model MODEL \
    --train \
    --data DATA_PATH \
    --iters 600

# 使用配置文件
mlx_lm.lora --config config.yaml
```

## fuse - 模型融合

```bash
# 融合 LoRA 适配器
mlx_lm.fuse --model MODEL --adapter-path ADAPTERS

# 导出为 GGUF 格式
mlx_lm.fuse --model MODEL --export-gguf

# 上传融合后的模型
mlx_lm.fuse \
    --model MODEL \
    --upload-repo your-username/fused-model \
    --hf-path ORIGINAL_MODEL
```

## cache_prompt - 提示词缓存

用于缓存长提示词，加速重复查询：

```bash
# 缓存提示词
cat long_text.txt | mlx_lm.cache_prompt \
    --model Qwen/Qwen2.5-7B-Instruct \
    --prompt - \
    --prompt-cache-file cache.safetensors

# 使用缓存
mlx_lm.generate \
    --prompt-cache-file cache.safetensors \
    --prompt "请总结以上内容"
```

## perplexity - 困惑度评估

```bash
mlx_lm.perplexity \
    --model MODEL \
    --dataset DATASET \
    --batch-size 4
```

## benchmark - 性能测试

```bash
mlx_lm.benchmark --model MODEL
```

## server - 启动模型服务器

```bash
# 启动服务器
mlx_lm.server --model MODEL

# 指定端口
mlx_lm.server --model MODEL --port 8080

# 允许外部访问
mlx_lm.server --model MODEL --host 0.0.0.0
```

## 管道用法

```bash
# 从文件读取提示词
echo "你好" | mlx_lm.generate --prompt -

# 处理多个提示词
cat prompts.txt | while read line; do
    mlx_lm.generate --prompt "$line" --max-tokens 100
done
```
