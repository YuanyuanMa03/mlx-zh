# 安装方法

## 前置准备

### 1. 检查系统

确保你的 Mac 搭载 Apple 芯片：

```bash
sysctl -n machdep.cpu.brand_string
```

输出应包含 "Apple M1/M2/M3/M4"。

### 2. 安装 Python

如果还没有安装 Python：

```bash
# 使用 Homebrew 安装
brew install python@3.11

# 验证安装
python3 --version
```

### 3. 创建虚拟环境（推荐）

```bash
# 创建虚拟环境
python3 -m venv mlx-lm-env

# 激活虚拟环境
source mlx-lm-env/bin/activate
```

## 安装 MLX-LM

### 使用 uv 安装（极速）⚡

**uv** 是一个超快速的 Python 包管理器，比 pip 快 10-100 倍。

```bash
# 安装 uv（如果没有安装）
curl -LsSf https://astral.sh/uv/install.sh | sh

# 使用 uv 创建项目并安装 mlx-lm
uv venv mlx-lm-env
source mlx-lm-env/bin/activate
uv pip install mlx-lm

# 或直接安装到当前环境
uv pip install mlx-lm
```

**为什么选择 uv？**
- ⚡ 极速安装（比 pip 快 10-100 倍）
- 🔒 自动处理依赖冲突
- 📦 内置虚拟环境管理
- 🚀 Rust 编写，性能卓越

### 使用 pip 安装（推荐）

```bash
pip install mlx-lm
```

### 使用 conda 安装

```bash
# 如果使用 conda
conda install -c conda-forge mlx-lm
```

### 从源码安装

```bash
# 克隆官方仓库
git clone https://github.com/ml-explore/mlx-lm.git
cd mlx-lm

# 安装
pip install -e .
```

## 安装额外组件

### 安装训练依赖

如果需要进行模型微调：

```bash
# 使用 uv
uv pip install "mlx-lm[train]"

# 使用 pip
pip install "mlx-lm[train]"
```

这将安装以下依赖：
- `datasets` - Hugging Face 数据集
- `tqdm` - 进度条显示

### 安装评估依赖

如果需要进行模型评估：

```bash
# 使用 uv
uv pip install "mlx-lm[evaluate]"

# 使用 pip
pip install "mlx-lm[evaluate]"
```

这将安装：
- `lm-eval` - 语言模型评估框架
- `tqdm` - 进度条显示

### 完整安装（包含所有功能）

```bash
# 使用 uv
uv pip install "mlx-lm[train,evaluate]"

# 使用 pip
pip install "mlx-lm[train,evaluate]"
```

## 验证安装

### 检查版本

```bash
python -c "import mlx_lm; print(mlx_lm.__version__)"
```

### 测试运行

```bash
# 简单测试
mlx_lm.generate --prompt "Hello"
```

如果成功生成文本，说明安装正确！

## 配置 Hugging Face（可选）

如果你想下载或上传模型到 Hugging Face：

```bash
# 安装 Hugging Face CLI
pip install huggingface_hub

# 登录
huggingface-cli login
```

## 常见安装问题

### 问题1：pip 安装失败

**错误信息**：`No matching distribution found for mlx-lm`

**解决方案**：
- 确认你在 macOS 上运行
- 确认使用的是 Apple 芯片 Mac
- 更新 pip：`pip install --upgrade pip`

### 问题2：MLX 版本不兼容

**错误信息**：`Requires MLX version ...`

**解决方案**：
```bash
pip install --upgrade mlx
```

### 问题3：权限错误

**解决方案**：
```bash
# 使用用户目录安装
pip install --user mlx-lm

# 或使用虚拟环境（推荐）
python3 -m venv mlx-env
source mlx-env/bin/activate
pip install mlx-lm
```

### 问题4：transformers 版本问题

**解决方案**：
```bash
pip install --upgrade transformers
```
