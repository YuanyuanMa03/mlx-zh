# 中文模型推荐

本页面推荐适合在 MLX-LM 上使用的中文大语言模型。

## 为什么选择专门的中文模型？

虽然通用模型（如 Llama）也能处理中文，但专门的中文模型在以下方面表现更好：

- 中文理解能力更强
- 中文生成质量更高
- 对中国文化和语境更了解
- 通常针对中文进行了优化

## 推荐：Qwen3.5 系列（最新）⭐

**Qwen3.5** 是阿里巴巴最新的 Qwen3 系列模型，专为"智能体 AI 时代"设计，在 2026 年 2 月发布。

### Qwen3.5-9B-MLX-4bit（强烈推荐）

```bash
# MLX Community 4-bit 版本
mlx_lm.generate --model mlx-community/Qwen3.5-9B-MLX-4bit --prompt "你好"
```

**特点：**
- 🆕 最新模型（2026年2月发布）
- 16GB 内存即可运行（实测 M1 16GB 可用）
- 支持 131k 上下文长度
- 内置"思考模式"用于复杂推理
- 出色的代码能力

**适用场景：**
- 日常对话和问答
- 代码生成和调试
- 复杂推理任务
- 智能体应用

### Qwen3.5-27B-4bit

```bash
mlx_lm.generate --model mlx-community/Qwen3.5-27B-4bit --prompt "你好"
```

**特点：**
- 更强的综合能力
- 工具使用：83%
- 代码能力：90%
- 推理能力：50%
- 需要 32GB+ 内存（M3 Ultra 实测约 15GB，38 tok/s）

**适用场景：**
- 专业级代码开发
- 复杂工具调用
- 高质量内容生成

## 按场景选择

### 日常对话助手（16GB 内存）

```bash
mlx_lm.chat --model mlx-community/Qwen3.5-9B-MLX-4bit
```

### 代码开发（32GB 内存）

```bash
mlx_lm.chat --model mlx-community/Qwen3.5-27B-4bit
```

### 创意写作

```bash
mlx_lm.generate \
    --model mlx-community/Qwen3.5-9B-MLX-4bit \
    --prompt "写一篇关于人工智能的科幻短文"
```
