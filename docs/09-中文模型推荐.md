# 中文模型推荐

本页面推荐适合在 MLX-LM 上使用的中文大语言模型。

## 为什么选择专门的中文模型？

虽然通用模型（如 Llama）也能处理中文，但专门的中文模型在以下方面表现更好：

- 中文理解能力更强
- 中文生成质量更高
- 对中国文化和语境更了解
- 通常针对中文进行了优化

## 推荐：Qwen 系列（通义千问）

**Qwen** 是阿里巴巴开源的大语言模型系列，在中文场景表现优异。

### Qwen2.5-7B-Instruct（推荐）

```bash
# MLX Community 4-bit 版本
mlx_lm.generate --model mlx-community/Qwen2.5-7B-Instruct-4bit --prompt "你好"

# 原始版本（需要先转换）
mlx_lm.generate --model Qwen/Qwen2.5-7B-Instruct --prompt "你好"
```

**特点：**
- 优秀的中文理解能力
- 适合日常对话和问答
- 7B 参数，16GB 内存可运行
- 4-bit 量化版本仅需约 4GB 内存

**适用场景：**
- 日常对话
- 中文问答
- 内容创作
- 文本摘要

### Qwen2.5-14B-Instruct

```bash
mlx_lm.generate --model mlx-community/Qwen2.5-14B-Instruct-4bit --prompt "你好"
```

**特点：**
- 更强的推理能力
- 适合复杂任务
- 需要 32GB+ 内存

**适用场景：**
- 复杂推理
- 专业领域问答
- 长文本生成

### Qwen2.5-32B-Instruct

```bash
mlx_lm.generate --model mlx-community/Qwen2.5-32B-Instruct-4bit --prompt "你好"
```

**特点：**
- 高质量输出
- 专业级性能
- 需要 64GB+ 内存

**适用场景：**
- 专业写作
- 复杂分析
- 高质量内容生成

## 代码生成：Qwen2.5-Coder

```bash
mlx_lm.generate --model mlx-community/Qwen2.5-Coder-7B-Instruct-4bit --prompt "用Python写一个快速排序"
```

**特点：**
- 专门针对代码生成优化
- 支持多种编程语言
- 中文代码注释友好

**适用场景：**
- 代码生成
- 代码解释
- 技术问答

## 按场景选择

### 日常对话助手

```bash
mlx_lm.chat --model mlx-community/Qwen2.5-7B-Instruct-4bit
```

### 代码助手

```bash
mlx_lm.chat --model mlx-community/Qwen2.5-Coder-7B-Instruct-4bit
```

### 创意写作

```bash
mlx_lm.generate \
    --model mlx-community/Qwen2.5-14B-Instruct-4bit \
    --prompt "写一篇关于人工智能的科幻短文" \
    --temp 0.9
```

### 专业问答

```bash
mlx_lm.generate \
    --model mlx-community/Qwen2.5-32B-Instruct-4bit \
    --prompt "详细解释量子纠缠的原理" \
    --temp 0.5
```

## 其他中文模型

```bash

```

### ChatGLM

```bash
# 需要自己转换
mlx_lm.convert --model THUDM/chatglm3-6b
```

### InternLM（书生）

```bash
# 需要自己转换
mlx_lm.convert --model internlm/internlm2-7b
```

## 模型下载速度

如果 Hugging Face 下载速度慢，可以使用镜像：

```bash
export HF_ENDPOINT=https://hf-mirror.com
mlx_lm.generate --model Qwen/Qwen2.5-7B-Instruct
```

## 转换自己的中文模型

如果你有其他中文模型：

```bash
mlx_lm.convert \
    --model YOUR_MODEL_NAME \
    -q \
    --mlx-path ./my_chinese_model
```

## 建议

1. **初学者**：从 Qwen2.5-7B-Instruct-4bit 开始
2. **开发者**：使用 Qwen2.5-Coder 进行代码相关任务
3. **专业用户**：根据内存选择尽可能大的 Qwen2.5 模型
4. **双语需求**：考虑 Gemma-2-9b-it
